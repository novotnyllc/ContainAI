# fn-25-81w.2 Add sysbox version display to cai doctor

## Description
Update `cai doctor` to display installed and bundled sysbox versions, and indicate when an update is available.

### Text output changes (`_cai_doctor()`)
In the "Sysbox Isolation" section, after checking sysbox availability, add:
```
  Installed version: 0.6.7+containai.20260126    [OK]
  Bundled version: 0.6.7+containai.20260126      [OK] Up to date
  # OR if update available:
  Bundled version: 0.6.8+containai.20260127      [WARN] Update available
                                                 (Run 'cai update' to upgrade)
```

### JSON output changes (`_cai_doctor_json()`)
Add to the "sysbox" section:
```json
"installed_version": "0.6.7+containai.20260126",
"bundled_version": "0.6.7+containai.20260126",
"needs_update": false
```

### Summary section
Update Sysbox status to indicate if update is available:
```
  Sysbox:                                        [OK] Ready (update available)
```

### macOS handling
Skip bundled version check on macOS since sysbox is inside Lima VM.
