# fn-18-g96.6 Shell completion for template names

## Description

Add template name completion to the shell completion scripts. This extends the completion system from fn-18-g96.5.

**Template completion:**
- Complete `--template` parameter with available template names
- Source: Directory names under `${XDG_CONFIG_HOME:-~/.config}/containai/templates/` that contain a Dockerfile
- Uses `_cai_list_templates` from fn-18-g96.3

**Integration with completion scripts:**

For bash:
```bash
--template)
    local templates_dir="${XDG_CONFIG_HOME:-$HOME/.config}/containai/templates"
    if [[ -d "$templates_dir" ]]; then
        local templates=()
        for dir in "$templates_dir"/*/; do
            [[ -f "${dir}Dockerfile" ]] && templates+=("$(basename "$dir")")
        done
        COMPREPLY=($(compgen -W "${templates[*]}" -- "$cur"))
    fi
    return
    ;;
```

For zsh:
```zsh
'--template[Use custom template]:template:->templates'
# ...
templates)
    local templates_dir="${XDG_CONFIG_HOME:-$HOME/.config}/containai/templates"
    if [[ -d "$templates_dir" ]]; then
        _values 'templates' ${templates_dir}/*/(:t)
    fi
    ;;
```
