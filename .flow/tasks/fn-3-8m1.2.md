# fn-3-8m1.2 Implement SYNC_MAP declarative config array

## Description
Create the `SYNC_MAP` array with all config items from spec lines 49-111.

**Format**: `"source:target:flags"`

**Flags**:
- `d` = directory
- `f` = file  
- `j` = initialize JSON with `{}` if empty
- `m` = mirror mode (`--delete`)
- `s` = secret (600 files, 700 dirs)
- `x` = exclude .system/ subdirectory

**Items** (see spec for full list):
- Claude Code: claude.json, credentials.json, settings.json, plugins/, skills/
- GitHub CLI: ~/.config/gh/
- OpenCode: ~/.config/opencode/, auth.json
- tmux: .tmux.conf, .tmux/, ~/.config/tmux/
- Shell: .bash_aliases, .bashrc.d/
- VS Code Server: extensions/, data/Machine/, data/User/mcp/, data/User/prompts/
- VS Code Insiders: same structure
- Copilot: config.json, mcp-config.json, skills/
- Gemini: google_accounts.json, oauth_creds.json, GEMINI.md
- Codex: config.toml, auth.json, skills/ (exclude .system/)
## Acceptance
- [ ] SYNC_MAP array defined with all items from spec

- [ ] Flags correctly applied (d/f/j/m/s/x)
