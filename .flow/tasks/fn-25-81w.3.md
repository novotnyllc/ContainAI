# fn-25-81w.3 Add sysbox update to cai update command

## Description
Add `_cai_update_sysbox()` function to `src/lib/update.sh` and wire it into `_cai_update_linux_wsl2()`.

### New function: `_cai_update_sysbox()`
- Check if sysbox is installed (return 0 if not - don't fail)
- Check if update is needed using `_cai_sysbox_needs_update()`
- If update needed:
  - Show what will be updated (installed -> bundled)
  - Prompt for confirmation (unless --force or CAI_YES=1)
  - Call existing `_cai_install_sysbox_wsl2()` or `_cai_install_sysbox_linux()`
- Supports `--dry-run`, `--force`, and `CAI_YES=1`
- Skip on macOS (sysbox is inside Lima VM)

### Wire into update flow
Add call to `_cai_update_sysbox()` in `_cai_update_linux_wsl2()` after dockerd bundle update.

### Update help text
Add sysbox to "What Gets Updated" section in `_cai_update_help()`.
